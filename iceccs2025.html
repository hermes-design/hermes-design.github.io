1. Loading a PRISM Model

1. Open a terminal: Launch a terminal window in the directory where your PRISM model (.prism file) is located.

2. Execute the following command:

prism model_name.prism

Replace `model_name` with the actual name of your PRISM file.

For example:

prism my_model.prism

This command loads the PRISM model into the PRISM environment.

2. Verifying Model Syntax

PRISM will automatically check the syntax of your model when you load it. If there are any syntax errors, PRISM will display error messages indicating the problematic lines. Correct these errors in your .prism file and reload the model.

3. Running an Experiment

To perform an experiment (e.g., calculate probabilities, verify a property), use the following command:

prism .prism -prop .props
  • model_name.prism: The name of the file containing your PRISM model.
  • properties_file_name.props: The name of the file containing the properties to be verified (optional).

For example:

prism my_model.prism -prop my_properties.props

If you don't have a properties file, you can specify the property directly on the command line:

prism my_model.prism -p "P=? [F state_A]"

This command checks the probability of reaching state "state_A" in the model.

Interpreting the results

PRISM will display the analysis results in the console. Results may include probabilities, mean times, counter-examples (if a property is not satisfied), etc.

Results can also be exported in various formats (e.g., text, XML) for further analysis.
