PRISM Guide

Loading a PRISM Model

  1. Open a terminal in the directory containing your PRISM model (.prism file)
  2. Execute the following command:
prism model_name.nm

Replace model_name with your actual model file name.

For example:

prism my_model.nm

This command loads the PRISM model into the environment.

Verifying Model Syntax

PRISM automatically checks your model's syntax. If there are errors, they will be displayed with line numbers. Fix these errors in your .prism file and reload the model.

Running Experiments

To run experiments (calculate probabilities or verify properties), use the following command:

prism model_name.nm -prop property_name
  • model_name.prism: The name of the file containing your PRISM model.
  • property_name: The name of the property to be verified (optional).

For example:

prism my_model.nm -prop my_properties.props

If you don't have a properties file, you can specify the property directly on the command line:

prism my_model.nm -p "P=? [F state_A]"

This command checks the probability of reaching state "state_A" in the model.

Interpreting the results

PRISM will display the analysis results in the console. Results may include probabilities, mean times, counter-examples (if a property is not satisfied), etc.

Results can also be exported in various formats (e.g., text, XML) for further analysis.
